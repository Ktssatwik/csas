Customer Sales Analytics System
Use Case
A retail company sells products across multiple cities.
The company stores customer, order, and product information in a database, but the management team wants an analytics system that can generate reports and insights.
You are required to build a Customer Sales Analytics Tool using SQL, Python, and Streamlit.
The tool should:
Store and query sales data
Analyze customer purchasing behavior
Generate insights and reports
Handle data processing errors
Provide an interactive Streamlit dashboard
Organize the project using clean Python code
